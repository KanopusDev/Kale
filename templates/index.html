Send Email Example

POST 
curl -X POST "https://api.yourcompany.com/yourname/welcome-email" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "recipients": ["user@example.com"],
    "variables": {
      "name": "John Doe",
      "company": "Acme Corp"
    }
  }'

Response

{
  "success": true,
  "message": "Emails sent successfully",
  "data": {
    "request_id": "req_123456",
    "sent_count": 1,
    "failed_count": 0
  }
}
