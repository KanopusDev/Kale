Frequently Asked Questions

How do I get started with the API?

First, create an account and configure your SMTP settings in the dashboard. Then generate an API key and follow our documentation for integration examples.

What are the rate limits?

Rate limits vary by plan. Free accounts can send up to 100 emails per day. Paid plans have higher limits starting at 10,000 emails per month.

Can I use my own SMTP server?

Yes! Kale works with any SMTP server including Gmail, SendGrid, Amazon SES, and your own mail server. Just configure the settings in your dashboard.

How do I track email delivery?

Email delivery status is tracked automatically. You can view delivery reports in your dashboard or query the API for detailed delivery information.

How do I report bugs or request features?

You can report bugs or request features on our GitHub Issues page. Please provide detailed information to help us assist you better.

Is Kale open source?

Yes! Kale is completely open source. You can view the source code, contribute, or fork the project on GitHub.
